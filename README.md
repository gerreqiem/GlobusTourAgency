Турагентство "ГЛОБУС" - Информационная система

Описание проекта

Полнофункциональная информационная система для управления туристическим агентством. Позволяет автоматизировать процессы бронирования туров, управления заявками клиентов и учета автобусов.

Функциональные возможности

Для гостей:

-Просмотр списка доступных туров

-Информация о турах (фото, название, страна, даты, цена, места)

Для авторизованных клиентов:

-Все возможности гостя

-Бронирование туров

Для менеджеров:

-Все возможности клиента

-Просмотр и фильтрация всех туров

-Управление заявками (подтверждение/отклонение)

-Управление автобусами

-Поиск и сортировка туров

Для администраторов:

-Все возможности менеджера

-Добавление/редактирование/удаление туров

-Полное управление системой

-Особенности интерфейса

Подсветка туров:

- Скидка >15% → золотой фон (#FFD700)

- Осталось <10% мест → розовый фон (#FFB6C1)

- Тур через <7 дней → жирный шрифт

Адаптивный дизайн

Цветовая индикация статусов заявок

Структура базы данных

Основные таблицы:

Users - пользователи системы

Countries - справочник стран

BusTypes - типы автобусов

Tours - информация о турах

Requests - заявки на бронирование

Связи:

One-to-Many: Country → Tours

One-to-Many: BusType → Tours

One-to-Many: User → Requests

One-to-Many: Tour → Requests

Установка и запуск

Предварительные требования:

Windows 10/11

.NET Framework 4.7.2 или выше

SQL Server 2012 или выше

Visual Studio 2019 или выше

Шаги установки:

Клонировать репозиторий:

git clone https://github.com/yourusername/touragency.git

Настроить базу данных:

Открыть SQL Server Management Studio

Выполнить скрипт Database/CreateDatabase.sql

Настроить строку подключения в DatabaseService.cs

Настроить приложение:

Открыть TourAgency.sln в Visual Studio

Восстановить зависимости NuGet

Изменить строку подключения в DatabaseService.cs

Запустить приложение:

Нажать F5 в Visual Studio

Или собрать релизную версию

Настройка подключения к БД:

private string connectionString = @"Server=ВАШ_СЕРВЕР;Database=TourAgency;Integrated Security=True;";
